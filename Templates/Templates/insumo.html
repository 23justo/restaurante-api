Obtener Solo un Insumo y sus detalles

http://23justo.pythonanywhere.com/api/v1/insumo/15

                            
        Cuerpo del JSON de peticion
            {
            }
        Headers:
            Authorization: Token 766bccdb99a24a303b70aa3602d94c009f552bf2
Nota: El numero 15 que se muestra en la URL es el pk del objeto deseado

Creacion de un Insumo

Metodo POST hacia URL:
http://23justo.pythonanywhere.com/api/v1/insumo/

                        
    Cuerpo del JSON de peticion
        {
            "nombre": "insertar aqui nombre de insumo",
            "tipo_insumo": 1
        }
    Headers:
        Authorization: Token 766bccdb99a24a303b70aa3602d94c009f552bf2
Nota: el tipo de insumo hace referencia a un id, para obtener dicho id consultar la seccion de Tipos de Insumo

Respuesta Metodo POST


                                    
Cuerpo del JSON de Respuesta
    {
        "id": 30,
        "nombre": "insumo 2.11",
        "tipo_insumo": 2
    }

Errores Comunes en peticiones:
1.
    "tipo_insumo": [
        "Invalid pk \"55\" - object does not exist."
    ]
    Verificar el id de tipo_insumo

2.
    {
        "nombre": [
            "This field is required."
        ],
        "tipo_insumo": [
            "This field is required."
        ]
    }
    Los parametros no se estan enviando a la vista
