Respuesta Metodo GET


El parametro offset es un indicador de cuantos registros se deben saltar, se debe de tomar en cuenta que la paginacion por defecto siempre sera de 10 objetos.
Para obtener la siguiente pagina de registros se debe de utilizar la url:
127.0.0.1:8000/api/v1/tipoinsumo/?offset=10

Creacion de un Tipo Insumo

Metodo POST hacia URL:
http://23justo.pythonanywhere.com/api/v1/tipoinsumo/

                        
    Cuerpo del JSON de peticion
        {
            "nombre": "insertar aqui nombre de tipo insumo",
            "producto": 1
        }
    Headers:
        Authorization: Token 766bccdb99a24a303b70aa3602d94c009f552bf2
Nota: producto es el id de un registro tipo Producto, para obtener este id consultar la seccion Productos

Respuesta Metodo POST


                                    
    Cuerpo del JSON de Respuesta
    {
        "id": 3,
        "nombre": "Ejemplo creacion",
        "producto": 1
    }

Edicion de un Tipo Insumo

Metodo PUT hacia URL:
http://23justo.pythonanywhere.com/api/v1/tipoinsumoupdate/Pan

Se debe ingresar el nombre del TipoInsumo para su modificacion

            "nombre": "insertar aqui nombre de insumo",
            "producto": 1
        }
    Headers:
        Authorization: Token 766bccdb99a24a303b70aa3602d94c009f552bf2
