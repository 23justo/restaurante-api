El parametro offset es un indicador de cuantos registros se deben saltar, se debe de tomar en cuenta que la paginacion por defecto siempre sera de 10 objetos.
Para obtener la siguiente pagina de registros se debe de utilizar la url:
http://23justo.pythonanywhere.com/api/v1/usuario/usuario/?offset=10

http://23justo.pythonanywhere.com/api/v1/usuario/usuario/1

                            
        Cuerpo del JSON de peticion
            {
            }
        Headers:
            Authorization: Token 766bccdb99a24a303b70aa3602d94c009f552bf2
Nota: El numero 1 que se muestra en la URL es el pk del objeto deseado

Crear un usuario

Metodo POST hacia URL:
http://23justo.pythonanywhere.com/api/v1/usuario/usuario/

                            
Cuerpo del JSON de peticion
    {
        "username": "",
        "nombre": "",
        "email": "",
        "user_type": null,
        "compras_modulo": false,
        "inventario_modulo": false,
        "facturacion_modulo": false
    }
Headers:
    Authorization: Token 766bccdb99a24a303b70aa3602d94c009f552bf2
Nota: El user_type debe ser uno de estos tres valores:
- 'Contador'
- 'Admin'
- 'Mesero'
