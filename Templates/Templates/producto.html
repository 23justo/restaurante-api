    Cuerpo del JSON de peticion

Respuesta Metodo GET


El parametro offset es un indicador de cuantos registros se deben saltar, se debe de tomar en cuenta que la paginacion por defecto siempre sera de 10 objetos.
Para obtener la siguiente pagina de registros se debe de utilizar la url:
http://23justo.pythonanywhere.com/api/v1/producto/?offset=10

Creacion de un Producto

http://23justo.pythonanywhere.com/api/v1/producto/

                        
    Cuerpo del JSON de peticion
        {
            "nombre": "insertar aqui nombre del producto",
            "precio": 100
        }
    Headers:
        Authorization: Token 766bccdb99a24a303b70aa3602d94c009f552bf2
