Listado de Ordenes

Metodo GET hacia URL:
http://23justo.pythonanywhere.com/api/v1/compras/orden/

                        
    Cuerpo del JSON de peticion
        {
        }
    Headers:
        Authorization: Token 766bccdb99a24a303b70aa3602d94c009f552bf2

Respuesta Metodo GET


El parametro offset es un indicador de cuantos registros se deben saltar, se debe de tomar en cuenta que la paginacion por defecto siempre sera de 10 objetos.
Para obtener la siguiente pagina de registros se debe de utilizar la url:
http://23justo.pythonanywhere.com/api/v1/compras/orden/?offset=10

Obtener Solo una Orden y sus detalles

Metodo GET hacia URL:
http://23justo.pythonanywhere.com/api/v1/compras/ordenproductodetalle/1

                            
        Cuerpo del JSON de peticion
            {
            }
        Headers:
            Authorization: Token 766bccdb99a24a303b70aa3602d94c009f552bf2
Nota: El numero 1 que se muestra en la URL es el pk del objeto deseado
